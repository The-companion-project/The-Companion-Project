****The Companion Project:****

​Cages don't cure urges.

​Traditional blockers treat you like a prisoner. Companion is a free, open-source browser extension that replaces cold firewalls with empathy to build actual willpower.
​

**The Cool Elements:**

**​The Emotional Mirror**: Instead of a "Blocked!" screen, it asks how you feel. It matches your mood (e.g., 80% Lonely) with hand-written, empathetic dialogue to talk you down.

**​The Trust Test**: It randomly drops its shield: "I trust you today." If you fail, it doesn't punish you. It forgives you and pulls you back up.

**​Designed to Die**: After a 365-day streak, it says a final goodbye and permanently uninstalls. You stay free because of character, not code.
